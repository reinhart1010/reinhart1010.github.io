Dynamic Programming: Fibonacci Sequence

/posts/2020/10/25/dynamic-programming-fibonacci-sequence.html

Berikut ini adalah jawaban terhadap salah satu tugas kuliah COMP6049 - Algorithm Design and Analysis di BINUS University (Universitas Bina Nusantara).

Fibonacci Sequence merupakan sebuah deret di mana setiap angka dalam derettersebut merupakan hasil penjumlahan antara kedua bilangan sebelumnya (kecuali 2 angka pertama, yakni 1 dan 1). Jika saya ingin menghitung bilangan Fibonacci ke-N, saya tentunya dapat melakukannya seperti ini:

FIB
Read N
IF N <= 2 THEN
    Return 1
ENDIF
Return FIB(N - 2) + FIB(N - 1)
END

Namun, jika program tersebut dijalankan dengan nilai N yang besar (seperti 100), program tersebut akan memakan waktu yang sangat lama. Mengapa?

Jika kita telusuri pseudocode tersebut, untuk memanggil FIB(100) kita harus memanggil FIB(99) dan FIB(98). Untuk memanggil FIB(99), saya harus memanggil FIB(98) dan FIB(97), sedangkan untuk memanggil FIB(98) saya harus memanggil FIB(97) dan FIB(96). Di sini kita dapat melihat bahwa perintah FIB(98), FIB(97), FIB(96), hingga FIB(1) dipanggil berulang-ulang dengan hasil yang sama (tidak ada pengacakan/randomization yang dilakukan oleh algoritma tersebut).

Masalah ini dapat diselesaikan dengan memodifikasi algoritma tersebut. Jika sebelumnya fungsi FIB(N) akan menghitung nilai secara rekursif, kali ini fungsi tersebut akan mengecek apakah hasil perhitungan sebelumnya sudah disimpan.

START
Set fib_array
DO index=1 to number_of_elements
    fib_array(index) = 0
ENDDO
FIB(100)
END

FIB
Read N
IF fib_array(N) != 0 THEN
    Return fib_array(N)
ENDIF
Set result
IF N <= 2 THEN
    result = 1
ELSE
    result = FIB(N - 2) + FIB(N - 1)
fib_array(N) = result
Return result
END

Teknik ini sering disebut sebagai Dynamic Programming, atau lebih spesifiknya lagi, proses Memoization. Proses ini ditujukan untuk menyimpan hasil perhitungan FIB(N) pertama yang akan dapat dipakai dalam perhitungan FIB(N) berikutnya. Teknik ini juga serupa dengan mekanisme caching yang sering dipakai dalam berbagai program dan software.
