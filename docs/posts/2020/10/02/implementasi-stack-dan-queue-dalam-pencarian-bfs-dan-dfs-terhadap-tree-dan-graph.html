Implementasi Stack dan Queue dalam pencarian BFS dan DFS terhadap Tree dan Graph

/posts/2020/10/02/implementasi-stack-dan-queue-dalam-pencarian-bfs-dan-dfs-terhadap-tree-dan-graph.html

Reinhart Previano Koentjoro October 2, 2020 | @binus @binus-socs #id-id #comp6049

Berikut ini adalah jawaban terhadap salah satu tugas kuliah COMP6049 - Algorithm Design and Analysis di BINUS University (Universitas Bina Nusantara).

Prinsip Stack dan Queue dapat diterapkan dalam proses pencarian suatu tree atau graph melalui dua jenis algoritma yang berbeda. Di sini, Stack dapat digunakan dalam algoritma traversal Depth-First Search (DFS). Sedangkan, Queue dapat digunakan dalam algoritma traversal Breadth-First Search (BFS).

Salah satu perbedaan utama antara DFS dengan BFS adalah, algoritma DFS akan berusaha untuk mencari dari sebuah subtree/subgraph/child sampai ke tingkat terdalam sebelum sebelum mencari di children lainnya, sedangkan BFS akan berusaha untuk mencari semua node dalam tingkat yang sama, sebelum mencari para node dalam tingkat berikutnya.

Proses DFS menggunakan Stack untuk menentukan prioritas urutan node yang dicari, di mana children dari sebuah node akan diprioritaskan untuk dicari daripada sepupu dari node itu sendiri. Di sini saya akan menyertakan contoh penerapan DFS dalam pencarian node H pada Gambar 3.1 dalam makalah https://informatika.stei.itb.ac.id/~rinaldi.munir/Matdis/2008-2009/Makalah2008/Makalah0809-054.pdf, yang dapat direpresentasikan dalam tabel berikut.

Posisi NodeStackSudah dikunjungi
AA-
AB-CA
BD-E-CA, B
DE-CA, B, D
EF-G-CA, B, D, E
FG-CA, B, D, E, F
FH-I-CA, B, D, E, F, G
HI-CA, B, D, E, F, G, H

Sedangkan, proses BFS menggunakan Queue untuk menentukan prioritas urutan node yang dicari, di mana seluruh node dalam satu tingkat tree yang sama akan diprioritaskan sebelum para node yang akan ada di dalam tingkat berikutnya (yakni seluruh children dari para node yang diproses terlebih dahulu). Saya juga akan menggunakan contoh tree yang sama untuk mendemonstrasikan proses BFS dalam tabel berikut.

Posisi NodeQueueSudah dikunjungi
AA-
AC-BA
BE-D-CA, B
CE-DA, B, C
DEA, B, C, D
EG-FA, B, C, D, E
FGA, B, C, D, E, F
GI-HA, B, C, D, E, F, G
HIA, B, D, E, F, G, H

Proses pencarian BFS dan DFS untuk graph tidak jauh berbeda dengan tree. Untuk menggunakan BFS dan DFS ke dalam pencarian Graph, Graph tersebut akan dikonversi dalam bentuk tabel Adjacency List yang juga digunakan dalam pembentukan Minimum Spanning Tree (MST). Demikian jawaban dari saya, Terima kasih.
